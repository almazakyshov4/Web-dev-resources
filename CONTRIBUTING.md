# Как внести свой вклад

Мы рады, что вы хотите помочь улучшить этот список ресурсов! Пожалуйста, следуйте этим рекомендациям.

## Процесс внесения изменений
1. Форкните этот репозиторий.
2. Создайте ветку для ваших изменений (`git checkout -b feature/amazing-resource`).
3. Добавьте ваши ресурсы в соответствующую категорию.
4. Сделайте коммит (`git commit -m 'Добавлен [Название ресурса]'`).
5. Запушьте изменения в ваш форк (`git push origin feature/amazing-resource`).
6. Откройте Pull Request.

## Правила
- Убедитесь, что ссылка рабочая.
- Добавьте краткое описание ресурса.
- Соблюдайте алфавитный порядок (по возможности).
