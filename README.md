# Ресурсы веб-разработки

Добро пожаловать в репозиторий ресурсов для веб-разработки! Здесь вы найдете полезные инструменты, библиотеки и руководства.

## Содержание
- [HTML](#html)
- [CSS](#css)
- [JavaScript](#javascript)

## HTML
- [MDN Web Docs - HTML](https://developer.mozilla.org/ru/docs/Web/HTML) - Полное руководство и справочник по HTML.
- [HTML5 Boilerplate](https://html5boilerplate.com/) - Профессиональный фронтенд-шаблон.
- [Can I use](https://caniuse.com/) - Таблицы поддержки современных веб-технологий в браузерах.

## JavaScript
- [MDN Web Docs - JavaScript](https://developer.mozilla.org/ru/docs/Web/JavaScript) - Полное руководство по JS.
- [JavaScript.info](https://learn.javascript.ru/) - Современный учебник JavaScript.
- [ES6 Features](http://es6-features.org/) - Обзор возможностей ES6.
